## Planet E-commerce

This app is build using ReactJS, mockbee which has following functionalities:
### Pages
 * Landing page
 * Products page
 * Cart page
 * Wishlist Page
 * Login Page
 * Logout Page

### Featured
 * Sort by price
 * Filter by category
 * Filter by rating
 * Filter by Price
 * Clearing all filters
